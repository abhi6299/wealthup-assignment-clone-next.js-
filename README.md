## Installation
1. Clone the directory
   ```
   git clone <repo_URL>
   ```
3. Navigate to the project directory
   ```
   cd <Project_directory>
   ```
4. Install Dependencies
   ```markdown
   npm i
   ```

## Usage
1. Start the development server:
```markdown
npm run dev
```
2. Open your browser:
```bash
  Open the browser and go to http://localhost:3000 to view the running UI.
```
